# Hopscotch

Hopscotch is a lightweight, pip-installable app that tracks alcohol drinking and collecting for the aficionado.

## Apps

### Dram
An app that stores drink information

## Requirements

### Python

All pip installable, included in .requirements.txt

mongo >= 2.0

### Web
